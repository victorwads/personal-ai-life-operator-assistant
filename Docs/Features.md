# Features

This document summarizes the capabilities currently present in the app.

## WhatsApp

- discover chats locally
- read recent messages
- send messages
- wait for unread chat activity
- search for chats that are not immediately visible
- manage archived or pending conversations

## Voice

- speak messages to the client
- ask the client for a response
- store pending answers locally
- show client voice history
- replay past voice events
- support a hands-free response flow

## Memory

- create memories
- retrieve memories
- search memories by similarity
- list stored memories
- delete memories when needed

## Sensitive data

- save sensitive records
- update sensitive records
- retrieve sensitive records by exact key
- search sensitive records
- list sensitive records
- delete sensitive records
- audit sensitive-data access

## Subjects

- create subjects
- update subjects
- resolve subjects
- cancel subjects
- inspect active subjects
- retrieve a single subject

## Nicknames

- list nicknames
- save nicknames
- delete nicknames

## Settings and control

- configure the MCP host and port
- control polling
- manage outgoing-message prefix and signature behavior
- request Accessibility permission
- inspect logs and debug views

## Debugging and observability

- server logs
- integration logs
- WhatsApp debug captures
- Accessibility debug trees
- server tool browser

## Current direction

The next capabilities on the roadmap include:

- LM Studio session control from inside the app
- remote/mobile operation
- richer runtime observability
- automated integration tests
- better session and event supervision
